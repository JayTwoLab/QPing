## QPing

> *Read this in other languages: [English](README.md), :kr: [한국어](README.ko.md)*

### 요약
- Ping 클래스. Qt5 지원.

### 노트 
- ICMP 프로토콜을 사용하는 함수를 부르지 않도록 구성하였습니다.
	- 저는 예전에 로소켓을 사용하는 유사한 클래스를 만든 적이 있습니다.
	- 하지만 그런 경우 사용자 계정의 권한 상읏이 필요합니다. 루트, 슈퍼유저, 어드민 등 정도의 권한이 필요하지요.
